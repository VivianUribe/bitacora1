Tema 1

Temas vistos en la materia pasada

Wireframe

En esta clase, utilizamos wireframes para planificar la estructura visual de una página antes de escribir el código HTML. Esto ayuda a organizar los elementos y mejorar la experiencia del usuario (UI - User Interface).

Header

El header, o cabecera, es una de las partes más importantes de una página web. Es el primer elemento que los usuarios ven al ingresar a un sitio, por lo que debe ser ordenado, claro y atractivo para causar una buena primera impresión.

Footer

El footer o pie de página es una sección frecuentemente subestimada en el diseño web, pero si se estructura y diseña correctamente, puede aportar valor, funcionalidad y confianza al sitio.

Hojas de estilo

La tipografía se ajusta con propiedades como el tipo, tamaño y estilo de la fuente. El color modifica tanto el texto como los fondos. Los filtros aplican efectos visuales, como desenfoque o brillo. La propiedad position controla cómo se posicionan los elementos, ya sea de manera estática, relativa, absoluta o fija. Finalmente, display define el comportamiento de los elementos en el flujo de la página, como bloque, en línea o en cuadrícula.

Rutas

Las rutas en HTML se utilizan para enlazar diferentes recursos o páginas dentro de un sitio web. Estas rutas pueden ser absolutas o relativas: Rutas Absolutas: Indican la ubicación completa del recurso, incluyendo el dominio. Ejemplo: a href="https://www.ejemplo.com/pagina"Ir a la página Rutas Relativas: Especifican la ubicación del recurso en relación a la página actual. Ejemplo: a href="/carpeta/pagina.html" Ir a la página Las rutas también se usan para enlazar a archivos locales (como imágenes o documentos) o a páginas internas dentro de un sitio web.

Especificidad

La especificidad es el mecanismo mediante el cual los navegadores determinan qué valores de propiedad CSS son más relevantes para un elemento y, por lo tanto, cuáles se aplicarán.

Imagenes

Las imágenes ayudan a los buscadores a indexear las paginas, brindan a los usuarios buenas experiencias de usuario ux y sirven de explicación grafica

Enlaces

La web funciona mediante links, que nos llevan a otras paginas, sitios web, imágenes etc, un link puede ser un texto, una imagen, un video etc.

Listas

En HTML, las listas son elementos que se utilizan para organizar y agrupar contenido relacionado en forma de elementos enumerados o con viñetas. Son muy útiles para mostrar datos de forma clara y ordenada.

Formularios

Los formularios permiten a los usuarios enviar datos a un servidor, la etiqueta form define un formulario y puede contener elementos como input, textarea y button.

Tablas

Es un conjunto estructurado de datos distribuidos en fiñas y columnas una tabla permite buscar con rapidez y facilidad valores entre diferentes tipos de datos que indiquen algun tipo de conexion.

Animación

Una animación en CSS3 es una técnica que permite aplicar movimiento, cambios de estilo o efectos visuales a los elementos HTML de una página web, sin necesidad de JavaScript.

Tema 2

Conceptos

¿Qué es una página web?

Una página web es un documento digital accesible en internet que contiene texto, imágenes y otros elementos interactivos, y que se visualiza mediante un navegador.

UX

Es cómo se siente una persona al interactuar con un producto, servicio o sistema.

Prototipado

Es como crear un boceto o una versión preliminar de una página web antes de desarrollarla completamente.

    Unidades de medida estáticas y relativas

    Unidades estáticas o absolutas: Son fijas, no cambian sin importar el dispositivo o el tamaño de la pantalla. Siempre miden lo mismo. Por ejemplo, si algo mide 200 píxeles, será 200 píxeles en una computadora, en un celular o en una tablet.

    Unidades relativas: Son flexibles porque dependen de algo más. Su valor cambia según el contexto: el tamaño del contenedor, de la fuente, o del propio viewport (la ventana visible de la página). Gracias a esto, permiten que el diseño se adapte a diferentes pantallas.

    Tema 6

    Investigación (día que no hubo clase)

    Color animation: Permite animar cambios de color en elementos, como el color de fondo (background-color), el color del texto (color) o los bordes. Se utiliza para generar transiciones visuales atractivas y resaltar interacciones con el usuario.

    Delay (animation-delay): Define el tiempo de espera antes de que la animación comience a ejecutarse. Sirve para controlar el momento exacto en que la animación inicia, útil cuando quieres coordinar varias animaciones o generar un efecto de secuencia.

    Animation-timing-function: Controla la velocidad de la animación a lo largo de su duración. No es lo mismo que la animación vaya siempre a la misma velocidad que comience rápido y termine lento. Algunos valores comunes son linear, ease, ease-in, ease-out y ease-in-out.

    Animation-direction: Indica en qué dirección se ejecuta la animación. Puede ser hacia adelante (normal), hacia atrás (reverse), alternando entre ambas direcciones (alternate), o alternando en reversa (alternate-reverse). Esto permite crear efectos de ida y vuelta sin necesidad de duplicar la animación.

    Animation-fill-mode: Define cómo se aplican los estilos de la animación antes y después de que esta ocurra. Por ejemplo, con forwards el elemento mantiene el último estado de la animación, con backwards toma el estilo inicial durante el retraso, y con both se aplican ambos comportamientos.

    Tema 7

    Codigo clase

    em: Es una unidad que depende del tamaño de letra de su elemento padre, se usa cuando quieres que los elementos crezcan o disminuyan según el tamaño de su contenedor..
    Si el padre tiene font-size: 20px y el hijo tiene font-size: 2em, entonces el hijo mide 40px porque 20px × 2 = 40px.

    rem: Es parecido al em, pero en lugar de depender del padre, siempre toma como referencia el tamaño del html. Por defecto, el navegador pone html { font-size: 16px; } Si usas font-size: 2rem, eso será 32px sin importar en qué contenedor esté.

    vh (viewport height): Es un porcentaje del alto de la ventana del navegador, se usa mucho para secciones a pantalla completa, ejemplo:
    100vh = toda la altura de la pantalla. 50vh= la mitad de la altura de la pantalla.

    vw (viewport width): Es un porcentaje del ancho de la ventana del navegador, se usa para que los elementos se adapten al ancho de cualquier dispositivo, ejemplo:
    100vw = toda el ancho de la pantalla. 75vw= del ancho de la pantalla.

    nav ul { text-align: center; }: Dentro de esa condición, el contenido de la lista (ul) se centra horizontalmente. Así, los enlaces del menú (Inicio, Acerca de, Contacto) se ven más ordenados en pantallas pequeñas.
